# Notation

**Notation** is a web app that helps you organize and manage your tasks locally

## Features
- Add tasks
- Delete tasks
- Mark tasks as Done
- Pin tasks
- Search tasks

## Installation
1. Clone the repository
```bash
git clone https://github.com/atomi19/notation.git
```
2. Navigate to the project directory
```bash
cd notation
```
3. Open the _index.html_ file
